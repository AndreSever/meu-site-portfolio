# Meu Site Portfolio

Portfolio profissional estratégico para consolidar e expor minha trajetória de carreira.

## Objetivo

Este projeto tem como propósito consolidar e expor de forma clara e estratégica minha trajetória profissional, destacando:

- **Conquistas Profissionais**: Marcos importantes e resultados alcançados ao longo da carreira
- **Realizações & Projetos**: Detalhamento de iniciativas pessoais e corporativas de alto impacto
- **Entregáveis**: Demonstração prática de escopos, produtos e soluções desenvolvidas

## Tecnologias

- **Next.js 14**: Framework React com App Router
- **shadcn/ui**: Componentes UI profissionais e acessíveis
- **TailwindCSS**: Estilização utility-first
- **TypeScript**: Tipagem estática para robustez
- **Framer Motion**: Animações suaves e interativas

## Estrutura do Projeto

```
meu-site-portfolio/
├── src/
│   ├── app/           # Páginas e rotas (App Router)
│   ├── components/    # Componentes reutilizáveis
│   ├── lib/           # Utilitários e configurações
│   └── styles/        # Estilos globais
├── public/            # Arquivos estáticos (imagens, ícones)
├── docs/              # Documentação adicional
└── README.md          # Este arquivo
```

## Seções do Site

- **Sobre Mim**: Apresentação pessoal e perfil profissional
- **Conquistas**: Marcos importantes e resultados alcançados
- **Projetos**: Realizações detalhadas com impacto
- **Entregáveis**: Produtos e soluções desenvolvidas
- **Contato**: Links para redes sociais e contato

## Instalação

```bash
# Clonar repositório
git clone https://github.com/AndreSever/meu-site-portfolio.git

# Entrar no diretório
cd meu-site-portfolio

# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

## Uso

```bash
# Desenvolvimento
npm run dev

# Produção
npm run build
npm start
```
